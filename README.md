# Pharmacy ERP System 💊

نظام متكامل لإدارة الصيدليات مصمم لتسهيل عمليات البيع، إدارة المخزون، وتتبع حركة الأدوية بكفاءة عالية. النظام مبني لتلبية متطلبات الأعمال التجارية عبر واجهات مستخدم حديثة وقاعدة بيانات محلية سريعة.

## 🛠 التقنيات المستخدمة
* **لغة البرمجة:** Java
* **واجهة المستخدم:** JavaFX
* **قاعدة البيانات:** SQLite (عبر SQLite JDBC)

## ✨ الميزات الأساسية
* **إدارة المخزون (Inventory Management):** تتبع صلاحية الأدوية والكميات وتنبيهات النواقص.
* **إدارة المبيعات والمشتريات:** تسجيل فواتير الزبائن والموردين، وإدارة المرتجعات.
* **إدارة التأمين (Insurance):** دعم مبيعات شركات التأمين وتتبع الديون.
* **إدارة الصلاحيات (Role-Based Access):** نظام تسجيل دخول آمن يدعم أدوار مختلفة للمستخدمين (مدير، صيدلاني).
* **تتبع الورديات (Shift Management):** فتح وإغلاق الصناديق المالية لكل موظف.

## 🚀 كيفية تشغيل المشروع محلياً

### 1. المتطلبات الأساسية (Prerequisites)
تأكد من تنصيب البرامج التالية على جهازك:
* Java Development Kit (JDK 11 أو أحدث).
* JavaFX SDK (مطابق لنسخة الـ JDK).
* إضافة مكتبة الاتصال بقاعدة البيانات: `sqlite-jdbc`.

### 2. إعداد قاعدة البيانات (Database Setup)
نظراً للحفاظ على استقرار البيانات، لا يتم تضمين ملف قاعدة البيانات الحي `*.db` في هذا المستودع. لتهيئة قاعدة البيانات:
1. اذهب إلى مجلد `database` (أو مسار تخزين قاعدة البيانات في المشروع).
2. قم بتنفيذ ملف المخطط `schema.sql` (أو `كود قاعدة البيانات.txt`) باستخدام أي برنامج إدارة قواعد بيانات SQLite (مثل DB Browser for SQLite) لإنشاء الجداول والفهارس المطلوبة.
3. احفظ ملف قاعدة البيانات الناتج بالاسم الذي يستدعيه كود الاتصال في Java.

### 3. تشغيل المشروع
* قم باستيراد المشروع إلى بيئة التطوير الخاصة بك (IntelliJ IDEA / Eclipse / NetBeans).
* تأكد من ربط مكتبات JavaFX ومكتبة `sqlite-jdbc` بمسار بناء المشروع (Build Path).
* قم بتشغيل الفئة الرئيسية (Main Class) لبدء التطبيق.
